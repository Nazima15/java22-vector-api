# Java 22 Vector API Report  

이 저장소는 Java 22에서 새롭게 도입된 **Vector API**에 대한 보고서를 포함합니다.  
Vector API는 CPU의 SIMD(Vector) 명령어를 활용하여 **연산 성능을 향상**시키는 기능입니다.
